# Global Freelancers Data Cleaning, Analysis & Dashboard Project

## Project Overview

This project analyzes a global dataset of freelancers to uncover insights about their skills, experience, demographics, and professional profiles.
The work includes **data cleaning**, **data transformation**, and the creation of an **interactive Excel dashboard** using PivotTables and PivotCharts.

The goal was to turn an unstructured dataset into a fully cleaned, standardized, and visually insightful report.

---

## Tools Used

* Microsoft Excel
* PivotTables & PivotCharts
* Slicers for interactivity

---

## Data Cleaning & Preparation

* Standardized fields such as country, skills, domain, and gender
* Removed duplicates and corrected inconsistent entries
* Trimmed extra spaces and applied consistent number formats
* Grouped years of experience into brackets
* Detected blank or missing values in several columns and kept them as blanks rather than replacing them, to reflect the dataset exactly as provided.


---

## Analysis & Dashboard Development

### PivotTables and PivotCharts

Built PivotTables to explore:

* Primary skills
* Gender distribution
* Domains of work
* Activity status
* Hourly rates
* Ratings
* Country and language patterns

### Interactive Dashboard

Created a dynamic dashboard using PivotCharts and slicers to filter by:

* Primary skill
* Status (Active/Not Active)

Dashboard visuals highlight:

* Predominant gender per domain
* Hourly rate distribution
* Average ratings
* Experience brackets

## Key Insights

* Domains with the highest hourly rates were Cybersecurity and Web Developement
* Most freelancers have between 0 and 5 years of experience
* English is the most widely spoken language
* Active freelancers are charging more than Inactive freelancers used to charge

## Conclusion

The dataset was transformed from messy and inconsistent to fully analysis-ready.
The final interactive dashboard provides a clear and dynamic way to explore global freelance trends.
